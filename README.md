# StepCounterApp

With this app users can compete according to the steps that they took within competition time. All users' step datas are added on scoreboard and the first couple of users can win some special awards.

## Libraries
**Firebase**

**FirebaseDatabase**

**FirebaseAuth**

**FirebaseAnalytics**

**UIKit**

**CoreMotion**

**StoreKit**
